# 🎓 SkillSync: Campus Peer-to-Peer Skill Exchange Platform

**SkillSync** is a centralized, peer-to-peer web platform that enables college students to exchange skills and knowledge through a secure, zero-cost barter system. It bridges the gap between students who want to learn new skills and those who are willing to teach them, fostering a collaborative learning environment on campus.

## 🚀 Key Features

- 📝 **Centralized Discovery:** Students can easily post, browse, and filter available skills by category (Tech, Arts, Academic, Languages).
- 🤝 **Smart Swap Mechanism:** Users can send personalized swap requests to exchange knowledge (e.g., "I teach Python, you teach Guitar").
- 🔒 **Privacy-First Connection (USP):** Contact details remain completely hidden until the swap request is mutually accepted, ensuring a spam-free and secure environment.
- 💸 **Zero-Cost Learning:** A pure knowledge barter system where no money is exchanged.
- 🛡️ **Secure Authentication:** Built-in user registration, login, and password hashing for secure access.

## ️ Technology Stack

- **Backend:** Python, Django Framework (MTV Architecture, ORM, Security)
- **Frontend:** HTML5, CSS3, JavaScript, Bootstrap 5 (Responsive UI)
- **Database:** SQLite (Relational Database)
- **Version Control:** Git & GitHub

## 🔄 How It Works

1. **Register & Login:** Users create secure accounts.
2. **Post a Skill:** Users list the skills they can teach or want to learn.
3. **Browse & Discover:** Explore skills posted by peers on the campus.
4. **Send Request:** Interested users send a personalized swap request.
5. **Accept/Decline:** The skill owner reviews the request.
6. **Connect:** Upon acceptance, the system securely reveals contact details for offline collaboration.
